🧮 How to Build a Modern HTML Calculator

This tutorial will guide you step-by-step to create a stylish, responsive calculator using HTML, CSS, and JavaScript. Perfect for mobile and web, even if you code on your Android phone.

🔧 Step 1: Create the HTML Structure

First, create a <div> with the class calculator and place an input field and button layout inside it.

<div class="calculator">
  <input id="resultMain" readonly />
  <!-- Buttons will be added here -->
</div>

🎨 Step 2: Style the Calculator with CSS

Add gradients, padding, shadows, and mobile responsiveness to make the calculator visually attractive. Below is a sample:

body {
  font-family: 'Segoe UI';
  background: #f8f9fa;
  padding: 20px;
}
.calculator {
  background: linear-gradient(135deg, #667eea, #764ba2);
  padding: 20px;
  border-radius: 20px;
  max-width: 350px;
  margin: auto;
  box-shadow: 0 20px 40px rgba(0,0,0,0.3);
}
.calculator input {
  width: 100%;
  height: 60px;
  font-size: 1.5em;
  text-align: right;
  padding: 0 15px;
}

⚙️ Step 3: Add JavaScript Logic

Use simple functions to evaluate input values, clear input, and show results:

function appval(val, target) {
  document.getElementById(target).value += val;
}
function clr(target) {
  document.getElementById(target).value = '';
}
function cal(target) {
  try {
    let output = eval(document.getElementById(target).value);
    document.getElementById(target).value = output;
  } catch {
    document.getElementById(target).value = "Error";
  }
}
